PillPal 💊 - Smart Medication Management & Pill Identification Mobile App

📌 Overview - PillPal is an Android mobile application designed to help users manage medications more effectively and safely. It combines image recognition, OCR (Optical Character Recognition), and Firebase services to support pill identification and prescription understanding.

This project was developed as part of my final-year BSc Software Engineering dissertation project.

🚀 Key Features
🔍 Pill Identification (Image-Based Recognition)- Users can capture or upload images of pills for identification support.
📄 Prescription OCR Scanning(Future updates) -Extracts text from prescriptions using OCR technology.
🔔 Medication Reminders - Helps users manage daily medication schedules.
🔐 Firebase Authentication - Secure login and user management using Firebase.
☁️ Cloud Integration - Stores and manages user-related data using Firebase backend services.

🧠 Project Goals

The main goal of PillPal is to improve medication safety and adherence by providing users with:

Easy pill identification
Prescription understanding support
Digital medication tracking
